```
项目部署启动方式：
1.下载项目到本地，使用idea2022之前的版本打开(2024经过测试无法使用，2023有时能够使用)
2.打开项目后等待maven下载对应的依赖包并构建项目
3.将项目中的数据库文件web.sql导入到mysql数据库，最好新建一个名为web的数据库
4.在src/main/resources/ApplicationContext.xml中的15-20行配置数据库连接信息，将数据库名，账号，密码修改为自己的。
5.在idea右侧点击数据库，添加mysql数据源，连接idea的数据库，配置账号密码
6.打开项目模块设计
  添加JPA，并在下方选择hibernate作为提供程序
  添加spring，点击加号添加xml配置文件
  添加web，添加xml配置文件。
7.更换项目的tomcat为本地版本，实测8，9，10版本都可使用，更早版本未测试。
8.启动项目，进入login.jsp界面代表启动成功。
```
